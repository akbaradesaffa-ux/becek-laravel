# CHANGELOG - Manage Locations CRUD

Update ini memperjelas fitur CRUD di halaman Manage Locations.

## Ditambahkan
- Tombol **Detail** sebagai fungsi Read data lokasi.
- Modal detail lokasi lengkap: kode, nama, kategori, area, harga, operasional, rekomendasi, total favorit, foto tambahan, fasilitas, dan link Google Maps.
- Panel ringkas Create, Read, Update, Delete di bagian atas Manage Locations.
- Label kolom aksi diubah menjadi **Aksi CRUD**.

## Tetap Dipertahankan
- Tambah lokasi.
- Edit lokasi.
- Hapus lokasi dengan konfirmasi.
- Search lokasi.
- Upload foto utama dan foto tambahan.
- Hapus foto tambahan.
- Fasilitas.
- Rekomendasi bintang.
